## Manual installation

**Clone project**

```cmd
git clone https://github.com/uuqkun/nexus-node-client.git
```

```cmd
cd nexus-node
```

**Install dependencies**
```cmd
npm install
```

**Running locally**
```cmd
npm run dev
```
